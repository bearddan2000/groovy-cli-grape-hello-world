# groovy-cli-grape-hello-world

## Description
Demonstrates groovy's in-built build tool `grapes`. Still requires either `gradle` or `maven` to include groovy library on classpath.

## Tech stack
- gradle
- groovy
    - grapes
    - grab

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
